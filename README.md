# share-mods-1.21.11
In deze omschrijving ga ik je uitleggen hoe je de mods download om op de server te kunnen gaan!

## Stap 1:
Om de CurseForge app te downloaden ga je naar deze link:<br />
[Download CurseForge link](https://www.curseforge.com/download/app)

Klik dan daarna op `Download Standalone`.

Als de `CurseForge Windows - Installer.exe` is gedownload open je het bestand.
Klik hier op verder bij alle stappen die je langs ziet komen. Hier hoef je niks aan te passen.

Open hierna de CurseForge app.

Kies hier het spel minecraft, klik dan op `Standard` bij de setup CurseForge modding folder.
Daarna word je gevraagd hoe je minecraft kan launchen. Kies hier voor de optie `Launch with CurseForge`

Log na deze stappen in met je minecraft/microsoft account waar je minecraft account is mee gekoppeld.

## Stap 2:
Download de `LyTiRi server + shaders-1.zip` uit mijn Github:<br />
[Mijn github link](https://github.com/R-Dam2005/share-mods-1.21.11)

*Dit is het profiel waar je op gaat spelen waar alle mods etc. al in zit!*

(Klik op de `LyTiRi server + shaders-1.zip` en klik daarna rechts in het beeld op de download knop. Deze staat tussen het icoontje van 2 blokjes en het icoontje van een potlood)

## Stap 3:
Ga terug naar CurseForge waar je bent ingelogd en voeg het profiel toe:

1. Kies in de CurseForge app `Minecraft`.
2. Ga naar het tabje `My Modpacks`.
3. Klik hier op `Import` en kies `Import Profile .zip`.
4. Ga naar de folder waar je het profiel in hebt gedownload en kies deze!<br />(de `LyTiRi server + shaders-1.zip`)
5. Nu heb je een profiel aangemaakt waar de mods inzitten!
6. Ga nu met je muis over het profiel heen en klik op `PLAY`!
